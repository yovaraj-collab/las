Q:Why made this? 
A: Many times, client wants immediate answer on the liquid level from the logged data, 3D view of MIT data, wants to witness the data of QP, Temp, ILS, CCL, Depth checking, and rest. So, thought of making life easy for the technical sales rep or anyone holding the .LAS file to respond immediately to client with their questions meanwhile waiting log analyst to show-up. No need any dongles to turn on the app. It's just 200kb web app hosted on trustable Github platform. All the mathematical codes are open source and no proprietary information in developing this viewer. Again, it is just a LAS Viewer to manage client's smallest requests!

-Supports all curves recorded in depth.
-Supports MIT24F
-supports LAS2 and LAS3 for far
-surprisingly only less than 200kb HTML page powered by Java, CSS canvas.
-ability to import step sizes 1,5,10 to load faster when rendering data.
